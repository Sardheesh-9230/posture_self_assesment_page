# 🧠 Posture Self-Assessment Using Deep Learning

A comprehensive, interactive educational webpage covering the research and applications of **deep learning–based posture self-assessment** — from real-time pose estimation to ergonomic risk analysis.

🌐 **Live Site:** [sardheesh-9230.github.io/posture_self_assesment_page](https://sardheesh-9230.github.io/posture_self_assesment_page/)

---

## 📌 Overview

This project presents a fully responsive, single-page educational resource built with **HTML & CSS** that covers:

- What posture self-assessment using deep learning is
- Step-by-step AI pipeline (from image capture to feedback)
- Key frameworks: MediaPipe, OpenPose, CNN, LSTM, Vision Transformers
- Real-world applications across healthcare, sports, ergonomics, and more
- Current challenges and future research directions
- **12 peer-reviewed journal references** with DOIs and direct links
